# CinematikInk Production Final

GitHub Pages-ready static website for CinematikInk.

## Before publishing
Update `https://your-domain.example/` in:
- `index.html`
- `robots.txt`
- `sitemap.xml`

## GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In repository settings, enable GitHub Pages.
4. Choose `Deploy from a branch` and select your main branch.

## Notes
- The site uses lightweight CSS/JS only.
- `assets/website-motion.webm` is an optimized motion asset.
- For best results, keep HTTPS enabled on your domain.
